default state:
Glass{arr1:[0......0]etc} - 10 arrays of 20 elements "0"

- cycle start with generation of new figure
    - figure fully viewed and centered when appear
- on each Step of figure moving its position rewrites
    - coordinates of figure - group of "adresses" in arrays grid, f.e. A1[Index],A2[Index] etc...
    - ability of figure to move checking ("0" Objects in array with Index+1 relatevely of figure bloks Indexes)
    - writes down coordinates of figures new position
    - old coordinates erased
- after each full Cycle, State's been rewrited with new data
    - full cycle - while figure is able to move down
    - when figure reachs lowest position cycle stops
    - change state with new default Glass
- when all the arrays gets "1" as an object with the same index by the end of cycle - line destroys
    - line rewrites with "0"
    - all the _top_ objects moves down with gravity
    - starts new cycle
    - change state with new default Glass
